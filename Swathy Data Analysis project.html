Over view¶

In this dataset,we will work with the advertising data of a marketing agency that predicts if a particular user will click on advertisement.¶

The data consists of 10 variables:

Daily Time spent on Time¶

Daily Internet Usage¶

Age¶

Area income¶

Ad Topic Line¶

Male¶

Country¶

City¶

Timestamp¶

Clicked on Ad.¶

The main variable we are interested is Clicked on Ad.This variable can have two possible outcomes:0 & 1 where 0 refers to case where an user didn't clicked on the Ad while 1 refers to the case where user clicks the advertisement. We will see if we can use the other 9 variables to accurately predict the value'Clicked on Ad variable'.
